# Nested Hubbard–Stratonovich Representations for Non-Gaussian Influence Functionals

Research project: exact nested HS construction for non-Gaussian influence functionals and the commuting-sector Hamiltonian of mean force.

## Quick Start

```powershell
./manage.ps1 install    # Install Python dependencies
./manage.ps1 sim        # Run all simulations
./manage.ps1 paper      # Compile LaTeX manuscript
```

## Structure

- `manuscript/tex/` — LaTeX source (revtex4-2, PRA style)
- `simulations/src/` — Python simulation scripts
- `simulations/results/figures/` — Generated figures
- `literature/` — Bibliography
